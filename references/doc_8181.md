# getRecoveryWorkerNum

**URL**: https://docs.dolphindb.cn/zh/funcs/g/getRecoveryWorkerNum.html

**来源**: DolphinDB 官方文档

---

getRecoveryWorkerNum
语法
getRecoveryWorkerNum()
详情
获取当前节点用于 chunk 恢复的工作线程数。
参数
无
返回值
整型标量。
例子
resetRecoveryWorkerNum(2)
getRecoveryWorkerNum()
// output: 2
相关函数：
resetRecoveryWorkerNum
