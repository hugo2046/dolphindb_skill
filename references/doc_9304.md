move
语法
move(X, steps)
详情
move
是
prev
和
next
的通用形式。
参数
X
可以是向量、矩阵或表；
steps
是一个整数或一个
duration
，表示移动多少位置或多少时间。
如果
steps
为正数，
X
向右移动
steps
个位置；
如果
steps
为负数，
X
向左移动
steps
个位置；
如果
steps
为 0，不改变
X
的位置；
如果
steps
为 DURATION，对应的
X
必须是行索引为时间类型的索引矩阵或者索引序列。
返回值
返回一个与输入
X
相同数据类型和形式的对象。
例子
x=3 9 5 1 4 9;
move(x,3);
// output: [,,,3,9,5]

move(x,-2);
// output: [5,1,4,9,,]
