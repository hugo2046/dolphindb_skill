# getInstrumentFixedRate

---

getInstrumentFixedRate
语法
getInstrumentFixedRate(instrument)
详情
根据输入的金融工具，获取该工具的固定端利率。
参数
instrument
INSTRUMENT 类型标量或向量，表示金融工具。
返回值
DOUBLE 类型标量或向量。
例子
swap =  {
    "productType": "Swap",
    "swapType": "IrSwap",
    "irSwapType": "IrFixedFloatingSwap",
    "version": 0, 
    "start": 2021.05.15,
    "maturity": 2023.05.15,
    "frequency": "Quarterly",
    "fixedRate": 0.02,
    "calendar": "CFET", 
    "fixedDayCountConvention": "Actual365",
    "floatingDayCountConvention": "Actual360",
    "payReceive": "Pay",
    "iborIndex": "SHIBOR_3M",
    "spread": 0.0005,
    "notional":["CNY", 1E8]
}
instrument = parseInstrument(swap)
getInstrumentFixedRate(instrument)

// output: 0.02
