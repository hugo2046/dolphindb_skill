lt

lt(X, Y)

详情
如果
X
和
Y

X
<
Y

如果
X
和
Y
都是集合，则检查
X

Y

参数
X
和
Y

X

Y

返回值
返回布尔类型的标量、向量、数据对或矩阵。
例子

// output
[1,0,0]

[0,0,1]
