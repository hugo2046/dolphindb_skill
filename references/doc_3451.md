# isYearStart

**URL**: https://docs.dolphindb.cn/zh/funcs/i/isYearStart.html

**来源**: DolphinDB 官方文档

---

isYearStart
语法
isYearStart(X)
详情
判断
X
是否为年初第一天。
参数
X
可以是 DATE, DATEHOUR, DATETIME, TIMESTAMP 或 NANOTIMESTAMP
                类型的标量或向量。
返回值
布尔标量或向量。
例子
isYearStart(2012.01.01);
// output: true

isYearStart([2012.01.01,2012.02.01]);
// output: [true,false]
