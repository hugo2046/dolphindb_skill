listMCPPrompts
语法
listMCPPrompts()
详情
查看所有已定义的 MCP prompt 模板。
返回值
一张表，包含以下字段：
name：STRING 类型，prompt 模板的名称。
description：STRING 类型，prompt 模板的描述。
title：STRING 类型，prompt 模板的 title。
lastModifyTime：TIMESTAMP 类型，最后一次修改时间。
publishTime：TIMESTAMP 类型，发布时间。
例子
listMCPTools()

message

title
lastModifyTime
publishTime
