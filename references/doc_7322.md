# isNumeric

**URL**: https://docs.dolphindb.cn/zh/funcs/i/isNumeric.html

**来源**: DolphinDB 官方文档

---

isNumeric
语法
isNumeric(X)
详情
判断
X
是否只包含数字。如果
X
中的所有字符都是数字，该函数返回 true，反之返回
                false。对于空字符串（STRING 类型的 NULL 值），该函数返回 false。
参数
X
是字符或字符串类型的标量、向量或表。
返回值
返回布尔类型，数据形式同 X。
例子
isNumeric("123456");
// output: true

isNumeric("1And1");
// output: false

isNumeric("10.05");
// output: false

isNumeric(string());
// output: false
