cbrt
语法
cbrt(X)
详情
返回
X
的立方根。
参数
X
可以是标量、数据对、向量、矩阵或表。
返回值
DOUBLE 类型，数据形式同
X
。
例子
cbrt(8);
// output
2

cbrt(8 12 16);
// output
[2,2.289428,2.519842]

cbrt(1..6$2:3);
0
